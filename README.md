# alfred-timestamp-convert
simple timestamp convert

## Demo
!["Demo"](demo.gif)

## Usage
- `ts now`
- `ts 1650637248`
- `ts 2022-04-22`
- `ts 2022-04-22 12`
- `ts 2022-04-22 12:12`
- `ts 2022-04-22 12:12:12`
